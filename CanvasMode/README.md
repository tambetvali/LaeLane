# Canvas Mode

[https://github.com/tambetvali/SimplyAboutInfinities/](https://github.com/tambetvali/SimplyAboutInfinities/tree/main/SimulationMode) - here, you get the 3 files for Laegna Numbers ([sheepcounter.json](https://github.com/tambetvali/SimplyAboutInfinities/blob/main/SimulationMode/sheepcounter.json), simplest), Laegna Wave Numbers ([sheepcounter.json](https://github.com/tambetvali/SimplyAboutInfinities/blob/main/SimulationMode/animation.json), moderate) and Laegna Octave Numbers ([octaves.json](https://github.com/tambetvali/SimplyAboutInfinities/blob/main/SimulationMode/octaves.json), advanced topic). Basic [multiline.md](https://github.com/tambetvali/SimplyAboutInfinities/blob/main/multiline.md) file at the repository root introduces Laegna Numbers and Laegna Wave Numbers in flat way, and belongs to package of several simple databases of Laegna numbers, most human-readable here: [flatlist](https://spireason.neocities.org/Playground/InferenceCounter/numberdatabase/flatlist/), which is flatlist folder for Users rather than Robots.

Currently, json and md files are also in our own project: [SrcAIPnP](https://github.com/tambetvali/LaeLane/tree/main/LaegnaLane/SrcAIPnP) contains multiline.md, all three json files, plus some manuals, htmls and examples to make sure it's also possible to figure out, what the names mean - multiline.md can have different introductions between project, but json files are identical at moment of writing, if the name is same - an AI might check size and hash, but also associate files through several locations and meanings.

Here, we will research several aspects about how to draw laegna numbers, and try to find out:
- I create similar datasets like Laegna Number Database, but for specific critical forms you need to use when visualizing numbers as geometric objects. These are simple objects you would need to be sure your geometric use cases are up to Laegna standards.
