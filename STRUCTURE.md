Here are what the essential folders and files in this project are and why:

- Folder: CanvasMode ~ here, we will add final results of work we do in LaneDatums.
  - ~ Lanedatums will contain all files needed for build, but CanvasMode would contain only the selected part of database needed to build Laegna Canvas.
- Folder: LaegnaLane ~ here, concept of Lane will be best explained and this is the folder of interproject documentation and semantic "API", also for an AI and human to understand *general point of the project*.
  - Folder: SrcAIPnP ~ here, one can find selection of Laegna Number Database files. [SAI/multiline.md](https://github.com/tambetvali/SimplyAboutInfinities/blob/main/multiline.md) and [Three .json files at SAI/SimulationMode](https://github.com/tambetvali/SimplyAboutInfinities/tree/main/SimulationMode) are the essential selection; those files are copied here along with more selected files.
  - File: Lane.md ~ central file of this repository, this is a human readable explanation and AI background for specific functions to draw lanes. Probably, the complete report about lanes will be finally, in the same folder, LaegnaLane.md, which will become the central file while Lane.md becomes the central specification and I will go through it and add bolds, math syntax or any other thing to beautify it.
  - File: STRUCTURE.md ~ local listing of files, which consist the Laegna Number Database, which is a little more powerful than the one under SimplyAboutInfinities, but it's less probable an AI would read it all into context, so it should extract structural data and other information from here, but use multiline.md and two or three jsons from SAI (SimplyAboutInfinities).
  - File: README.md ~ this will be the central introduction to topic of Laegna Lanes.
- Folder: LaneDatums ~ internal working folder: here we do all the intermediate work for the project, and leave draft files.
  - ~ this will contain everything about Laegna Lanes.
